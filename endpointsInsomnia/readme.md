# Importar Endpoints en Insomnia
Para aquellos que deseen tener todos los endpoints del proyecto en Insomnia de manera automática, pueden seguir los siguientes pasos:

    - Descarguen el archivo adjunto proporcionado
    - Abran Insomnia y seleccionen la opción Create
    - Dentro del menú, elijan la opción Import
    - Arrastren el archivo descargado previamente hacia Insomnia
    - Finalmente, hagan clic en Scan

Al realizar estos pasos, todos los endpoints necesarios para utilizar el proyecto desarrollado a lo largo del curso serán importados automáticamente a Insomnia.
